Название проекта:
Автоматический сборщик отчета по проекту

Что делает программа:
Скрипт запрашивает путь к папке проекта, проверяет ее существование
и автоматически создает текстовый отчет.

В отчет записываются:
- дата и время создания;
- имя пользователя Windows;
- имя компьютера;
- путь к проекту;
- список всех файлов и папок;
- количество файлов;
- количество папок;
- список файлов с расширениями .py, .html, .css, .js и .txt;
- проверка наличия README.md;
- проверка папок src, docs и tests.

Использованные команды:
echo
set /p
if exist
mkdir
dir
for
pause

Как запустить:
1. Открыть файл project_report.bat.
2. Ввести путь к папке проекта.
3. После завершения работы открыть файл reports\report.txt.
